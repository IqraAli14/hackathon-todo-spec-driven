# Hackathon Todo App - Spec Driven Development

An in-memory console todo application built using Spec-Driven Development (SDD) methodology.

## Quick Start

```bash
python -m src
```

## Features

- Add tasks with title and optional description
- List all tasks with status
- Mark tasks complete/incomplete
- Update task details
- Delete tasks

See `specs/001-todo-app/quickstart.md` for detailed usage.
